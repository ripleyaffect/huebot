Hue Server

About

It's an api for interacting with our very own Philips Hue bulbs

Usage

curl or just go to these urls in a browser to change the bulbs!

  • /hue/[value]
    where [value] is between 0 and 65535 (2^16 - 1) to change the color
  • /bri/[value]
    where [value] is between 0 and 255 (2^8 - 1) to change the brighness
  • /sat/[value]
    where [value] is between 0 and 255 (2^8 - 1) to change the saturation
  • curl -H "Content-Type: application/json" -d '{"sat":[sat_value], "bri":[bri_value],"hue":[hue_value]}' http://[local_ip]:[port]/set
    on the command line to change all at once
  • /color/[color_description]
    where [color_description] is a color with maybe some adjectives and stuff if you want.
  • /guess-color/[hue_value]
    to see what known color is closest to [hue_value]
